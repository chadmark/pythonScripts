# Robocopy PDF Move Script

## Overview

`Move-ConvertedPdfs_Robocopy.ps1` moves successfully converted PDF files to a destination folder using Robocopy.

The script reads the manifest generated by `pub_to_pdf_batch.py` and moves only files marked as successful conversions.

The original folder structure relative to the source root is preserved.

---

# Features

## Manifest-based filtering

The script reads:

```
conversion_manifest.csv
```

Only rows where:

```
status = success
```

are processed.

---

# Folder structure preservation

Example source structure:

```
C:\PublisherFiles\ClientA\doc1.pdf
C:\PublisherFiles\ClientB\doc2.pdf
```

Destination structure:

```
D:\Archive\ClientA\doc1.pdf
D:\Archive\ClientB\doc2.pdf
```

---

# Uses Robocopy

Robocopy is used because it is:

- faster than standard copy
- reliable for large batches
- resumable
- included with Windows

Microsoft documentation:

https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/robocopy

---

# Requirements

Windows system with:

- PowerShell
- Robocopy (included with Windows)

---

# Basic Usage

```
.\Move-ConvertedPdfs_Robocopy.ps1 `
  -ManifestCsv "C:\PublisherFiles\conversion_manifest.csv" `
  -SourceRoot "C:\PublisherFiles" `
  -DestinationRoot "D:\PDFArchive"
```

---

# Simulation Mode

To simulate the move process without moving files:

```
-WhatIf
```

Example:

```
.\Move-ConvertedPdfs_Robocopy.ps1 `
  -ManifestCsv "C:\PublisherFiles\conversion_manifest.csv" `
  -SourceRoot "C:\PublisherFiles" `
  -DestinationRoot "D:\PDFArchive" `
  -WhatIf
```

This prints the Robocopy commands that would run.

# Typical Workflow

## Step 1 — simulate conversions

```
python pub_to_pdf_batch.py "C:\PublisherFiles" --whatif
```

## Step 2 — simulate file moves

```
.\Move-ConvertedPdfs_Robocopy.ps1 `
  -ManifestCsv "C:\PublisherFiles\conversion_manifest.csv" `
  -SourceRoot "C:\PublisherFiles" `
  -DestinationRoot "D:\PDFArchive" `
  -WhatIf
```

## Step 3 — run real conversions

```
python pub_to_pdf_batch.py "C:\PublisherFiles"
```

## Step 4 — move PDFs

```
.\Move-ConvertedPdfs_Robocopy.ps1 `
  -ManifestCsv "C:\PublisherFiles\conversion_manifest.csv" `
  -SourceRoot "C:\PublisherFiles" `
  -DestinationRoot "D:\PDFArchive"
```

---

# Performance

Files are moved in chunks to avoid Windows command length limits.

Chunk size can be adjusted:

```
-ChunkSize 50
```

---

# Troubleshooting

## No files moved

Check the manifest for:

```
status = success
```

Simulated conversions (`success_simulated`) will not move files.

---

# Robocopy Exit Codes

Exit codes **0–7** are considered successful.

Reference:

https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/robocopy#return-codes

---

# Safety Notes

- Files are moved using `/MOV`
- Source PDFs are removed only after successful transfer
- `.pub` files are never modified
